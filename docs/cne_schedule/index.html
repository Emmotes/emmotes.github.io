Features:

  • Detects time ranges and replaces them with Discord timestamps.
  • Detects date lines and bolds them.
  • Wraps URLs in angle brackets to stop Discord embedding them.

If a time fails to convert, try one of these websites to make it manually:

  • https://r.3v.fi/discord-timestamps/
  • https://timestamp.dabric.xyz/
  • https://www.epochconverter.com/

Alternatively you can type @time in Discord and then press Tab. You can then type out a time and select the appropriate format.

Note that this method does not do short-time HH:MM for any day other than the present day. So to get that:

  1. Pick any date format for the date and time you want.
  2. Reveal the code by clicking the created timestamp and pressing the <> symbol.
  3. Change the very last letter to t.
  4. Delete the ` symbols before and after the code to turn it back into a timestamp.
